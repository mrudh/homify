# Homify UI/UX Design

Homify is a UI/UX design project focused on creating a user-friendly digital experience for home interior design support. The project explores how users can interact with a home-focused ecommerce platform through clear navigation, structured layouts, and visually engaging interface designs.

This repository contains the design portfolio and evaluation documents created for the Homify project.

## Project Overview

The aim of this project is to design a simple, modern, and user-centred interface for a home-related digital platform. The design focuses on making the user journey easy to understand, visually appealing, and practical for users who want to explore home design or personalisation options.

The project includes screen designs, design documentation, and a mini evaluation report to reflect on the usability and effectiveness of the proposed interface.
